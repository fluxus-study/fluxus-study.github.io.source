Javascript

나는 자바스크립트에 대해 얼마나 알고 있는가?
자바스크립트에서는 전부 다 객체. 함수도 객체. 

function getAdd(a, b){
  return a+b;
}

var addVal = getAdd(1,2);
console.log(addVal);


var getAdd = function(a, b){
  return a+b;
}

var addVal = getAdd(1,2);
console.log(addVal);
이런 식으로 함수를 객체로 만들어서 인자로 던지는것도 가능.

javascript 자료형

선언은 전부 var

  1. number (숫자) → var test2 = 100 ;
  2. string (문자열) → var test3 = “hahaha” ;
  3. boolean → var test4 = true ;
  4. object (객체) → var test6 = { };
  5. function (함수) → var test5 = function(){ }; //(람다함수)
  6. undefined → var test1 ; 값을 명시하지 않은 / 존재하지 않는 변수. null 이랑은 다름.

Function - 함수형.

함수 선언하는 3가지 방법. 


function addA(a, b){
  return a+b;
}

var addB = function(a, b){
  return a+b;
}

var addC = new Function('a','b','return a+b');

console.log(addA(1,2), addB(2,3), addC(3,4));

Function - 함수형.
